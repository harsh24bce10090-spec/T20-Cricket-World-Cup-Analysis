# 🏏 T20 Cricket World Cup Data Analytics Project

A complete **data analytics project** built using **Python and Power BI** to analyze T20 Cricket World Cup performance data.  
This project performs **data cleaning, preprocessing, analysis, and visualization** to generate meaningful insights about team performance and match statistics.

---

# 📌 Project Overview

The main goal of this project is to analyze T20 World Cup data and create an interactive dashboard for performance tracking and comparison.

The project helps in understanding:

- Team performance comparison
- Average runs scored
- Match trends
- Data-driven insights
- Interactive dashboard reporting

---

# 🚀 Technologies Used

- **Python**
- **Pandas**
- **NumPy**
- **Matplotlib**
- **Power BI**
- **GitHub Actions**

# ⚙️ Features

### Data Cleaning
- Removes null values
- Saves processed dataset

### Data Analysis
- Team-wise average runs
- Performance statistics

### Visualization
- Bar chart generation
- Exported graph images

### Power BI Dashboard
Interactive dashboard with:

- KPI Cards
- Team comparison charts
- Filters / slicers
- Performance insights

### Automation
GitHub Actions automatically runs analysis on every push.
